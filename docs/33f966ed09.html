<inner> asynchronousCommand

Executes an asynchronous command. For all intents and purposes this should be the same as running it on the command line. When the command is finished running the callback will be executed and any data sent to stdout, stderr, or any errors in attempting to run the command will be passed to it.
Parameters:
Name Type Description
commandString String The command you want to run.
workingDirectory String The path you want to start from.
callback Function Optional. A function to execute upon completion. The callback will be passed the error, stdout, and stderr arguments.
