Methods

process

Shifts commands off the command queue and executes them simultaneously. All commands will be given a callback as their only argument. The callback must be executed to allow the register to know when it is finished so it can emit 'command queue processed'. The callback itself will cause the register to emit 'command complete' once the command has finished executing.
